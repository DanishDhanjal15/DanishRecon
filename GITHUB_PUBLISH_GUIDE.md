   - Add .gitignore (Python template)

2. **Initial commit** (from root directory)
   ```
   git init
   git add .
   git commit -m "Initial commit: CyberRecon-Pro hackathon submission"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/CyberRecon-Pro.git
   git push -u origin main
   ```

3. **Verify on GitHub**
   - Check if all source files are visible
   - Verify no `results/`, `__pycache__/`, `.venv/` folders
   - Check that `.gitignore` is working

   - Tell judges the GitHub URL
   - Link in your presentation notes
